# Simple Tree

> Simple collapsible tree with lists 

[Live demo](http://revathskumar.github.io/simple-tree)

### Development

* Clone the repo

    `git clone git@github.com:revathskumar/simple-tree.git`

* Install npm and bower dependencies

    `npm install & bower install`

* start gulp server

    `gulp serve`
